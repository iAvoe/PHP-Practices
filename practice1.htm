Dice Rolling App

When the user loads the page, an AJAX call will happen that asks your PHP program, "dice_roll.php", for 5 "dice" values. These dice are random values, simulating a “roll” of 5 dice with 6 sides each. Your program will output HTML code that the AJAX call will display. Each die roll should be in a separate HTML element (maybe a <div>). Include BootStrap classes in your output to help make it more interesting and appealing.
Use rand(1,6) to generate the random integers.

Faux Math Library

In this practice exercise you will create a PHP function library to perform basic math operations, such as addition, subtraction, multiplication, and division. You will focus on code reusability and organization by encapsulating each operation in a separate function. Although these functions are very simple, they will give you practice at defining and using code, and separating processing code from creating a specific solution. Create a new PHP file named "math_functions.php" to contain all the functions for math operations. In a second file called "do_math.php" we call the functions and will create some output. Define the following functions: Function Name: add Parameters: Two numbers (num1, num2) Return Value: The sum of num1 and num2 Function Name: minus Parameters: Two numbers (num1, num2) Return Value: The result of subtracting num2 from num1 Function Name: times Parameters: Two numbers (num1, num2) Return Value: The product of num1 and num2 Function Name: divide Parameters: Two numbers (num1, num2) Return Value: The result of dividing num1 by num2 Note: Handle division by zero by returning NAN instead of performing the calculation In "do_math.php" include "math_functions.php", perform the following operations using the functions you created and generate a string with an HTML table. Each row must state the function being called, the inputs, and the result of the function call. If the result is NAN, do not show this to the user, instead create a meaningful message that explains the problem. - Calculate the sum of two numbers: 10 and 5. - Subtract 5 from 10. - Find the product of 7 and 3. - Divide 20 by 4. - Divide 15 by 0.
